# frontend-test

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Runs mocks server

```
npm run server
```

### Lints and fixes files

```
npm run lint
```

### Run test, lint watch for file changes to run tests again

```
npm run test:watch-all
```
